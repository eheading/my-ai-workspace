# 2026-05-02

## 今日完成
- 建立 `x-tracker/` 系統，追蹤 X.com 投資相關帳號並每日傳送 Telegram 摘要。

## 系統概覽
- **位置**：`x-tracker/`
- **技術棧**：twscrape（X.com 抓取）、OpenAI API（AI 分析）、python-telegram-bot（Telegram 推送）、APScheduler（排程）、SQLite（儲存）、Click（CLI）、Docker（部署）
- **核心流程**：每日定時抓取 → AI 篩選投資貼文 → 生成繁體中文摘要 → Telegram 推送

## 待使用者完成的設定步驟
1. 複製 `x-tracker/.env.example` 為 `x-tracker/.env`
2. 填入：`TELEGRAM_BOT_TOKEN`、`TELEGRAM_CHAT_ID`、`OPENAI_API_KEY`、`TWITTER_ACCOUNTS`
3. 使用 `python cli.py add <username>` 新增追蹤帳號
4. 執行 `python cli.py run` 測試完整流程
5. 部署：`docker compose up -d`

## 注意
- X.com 抓取使用 twscrape（需要真實 X.com 帳號 cookie）
- `.env` 含敏感資料，已加入 `.gitignore`
