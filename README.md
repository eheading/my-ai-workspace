# my-ai-workspace

這是一個個人 AI 工作空間，用來讓 GitHub Copilot 在不同對話中持續沿用同一套角色、記憶與工作方式。

## 這個倉庫包含什麼
- `AGENTS.md`：AI 助手的角色、工作方式與任務收尾規則
- `MEMORY.md`：長期記憶，保存穩定且可重用的資訊
- `memory/daily/`：每日或臨時記錄
- `memory/inbox.md`：尚未整理的想法、素材、待辦與草稿

## 使用方式
- 重要資訊盡量寫入文件，而不是只留在對話裡
- 穩定且長期有價值的內容整理進 `MEMORY.md`
- 當日進度與臨時內容記錄到 `memory/daily/`
- 尚未整理的內容先放到 `memory/inbox.md`

## 目標
讓這個倉庫成為一個可長期演化、輕量且實用的個人 AI 協作空間。
